Design System v2.0 (Ocean Breeze)

GLB-05

Atualização generalizada da estética corporativa TM em todos os artefatos de documentação.

  • Todos os arquivos commit.html do sistema e de projetos agora suportam dark mode perfeitamente sincronizado com o Ocean Breeze Design System usando os aliases CSS (--tm-bg, --tm-text).
  • Mapeamento da paleta antiga azul/roxa para a nova paleta oficial corporativa em escala Verde TM (#22c55e) em títulos, badges e backgrounds primários.
  • Emojis obsoletos de navegação em títulos ou dot notes (🚀/📄) totalmente banidos e substituídos por recursos vetoriais escaláveis estilo Lucide em inline SVG para alta resiliência e adequação em multi-temas.
  • Nova hierarquia tipográfica: Lora para títulos de páginas, DM Sans legível para blocos redigidos e dados em UI e IBM Plex Mono para numerações seriais e badges (GLB-XX).
  • Clean up do diretório NEXT APPS descontinuado, empurrando as atualizações para o fluxo 0 - NEXT APPS de forma unificada.
