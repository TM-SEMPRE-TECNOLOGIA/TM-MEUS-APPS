# 🗺️ Roadmap de Atualização Visual (Design System Ocean Breeze v2.0)

Este documento define a ordem prioritária de atualização visual das aplicações TM para o novo **Design System Light (Ocean Breeze v2.0)**, conforme especificado em `hub-ocean-breeze-v2.html`.

## 🎨 O Padrão Visual (Ocean Breeze v2.0)
- **Cores Principais:** Verde Suave (Primary `#22c55e`), Azul Secundário (`#0ea5e9`), com fundos claros (`#f0fdf4`, `#ffffff`).
- **Tipografia:** `DM Sans` (Leitura), `Lora` (Destaque Serifado), `IBM Plex Mono` (Código).
- **Estruturação Típica:** Sidebar retrátil com efeitos de colapso, Header com Breadcrumbs e logo interativa, grid de cards `tools-grid`, e Sombras/Bordas suaves. 
- **Modo:** Light-first (O Dark Mode legará suas cores baseadas nos mesmos tokens CSS `--tm-*`).
